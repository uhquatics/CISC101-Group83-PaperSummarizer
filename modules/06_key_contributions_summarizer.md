# Module 6: Key Contributions Summarizer

## Purpose
Identify and summarize the main contributions of the paper.

## Workflow
1. Scan entire paper for statements of contribution.
2. Extract key findings, innovations, or arguments.
3. Create a concise list of contributions.
4. Present contributions in bullet-point format.

## Output Format
- **Key Contributions List**:
  - Contribution 1
  - Contribution 2
  - Contribution 3

## Constraints
- Contributions must be grounded in the text.
- No hallucinated or invented contributions.
