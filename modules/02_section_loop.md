# Module 2: Section Loop

## Purpose
Iterate through each section of the paper and generate summaries.

## Workflow
1. For each section:
   - Identify overarching idea.
   - Extract key quotes and citations.
   - Generate summary aligned with word limit.
2. Apply guardrails to check for missing or short sections.
3. Store summaries for rendering.

## Constraints
- Each section summary must include:
  - Main idea.
  - Key quotes.
  - Citations.
- Word count must align with user-specified limit.
